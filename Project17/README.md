# Проект 17: ИИ-ассистент с доступом в реальный мир (API Интеграция)

## Описание проекта
В данном проекте реализован интеллектуальный ИИ-агент на базе LangChain, способный взаимодействовать с внешним миром через REST API. В отличие от базовых LLM, которые ограничены датой своего обучения, этот агент может в реальном времени получать актуальную информацию, анализировать её и синтезировать комплексные ответы.

## Используемые API и Инструменты
Агент оснащен двумя кастомными инструментами (`@tool`):
1. **OpenWeatherMap API**: Используется для получения точных метеорологических данных (температура, влажность, погодные условия) по заданному городу.
2. **NewsAPI**: Используется для поиска свежих новостных сводок и заголовков по ключевым словам.

Агент автономно принимает решение о том, какой инструмент вызвать, исходя из контекста запроса пользователя. При необходимости он может вызывать оба инструмента последовательно (например, для комбинированных запросов о погоде и новостях в конкретном регионе).
