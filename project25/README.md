# GenAI Chat Frontend

Это фронтенд-интерфейс для ИИ-ассистента, разработанный в рамках обучения. Приложение обеспечивает взаимодействие с FastAPI-бэкендом, поддерживая генерацию ответов в режиме реального времени (streaming).

## 🚀 Функционал
* **Интерактивный чат:** Поддержка истории диалога с пользователем.
* **Streaming Response:** Использование API для пошагового вывода текста модели, обеспечивающее отзывчивость интерфейса.
* **Современный стек:** Next.js (App Router), TypeScript, Tailwind CSS.
* **Архитектура:** Чистая структура с использованием кастомных хуков (`useChat`) для бизнес-логики и модульных компонентов.
