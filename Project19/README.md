# Project 19: AI Agent Debugging & Observability

## 📖 Описание проекта
Этот проект демонстрирует практические навыки интеграции систем мониторинга (Observability) в ИИ-агентов на базе LangGraph. Основная цель — использование **LangSmith** для отслеживания вызовов (tracing), выявления логических ошибок, зацикливаний агента и "галлюцинаций", вызванных плохим промптингом.

## 🚀 Стек технологий
* Python 3
* LangChain & LangGraph (фреймворк агентов)
* OpenAI API (gpt-4o-mini)
* **LangSmith** (мониторинг и трейсинг)

## 📂 Структура проекта
* `broken_agent.py` — Агент с намеренно внедренными дефектами логики (v1-broken). Демонстрирует `GraphRecursionError` и некорректную обработку 503 ошибок.
* `fixed_agent.py` — Исправленная версия агента (v2-fixed) с корректным системным промптом и "изящной деградацией" при сбоях инструментов.
* `debugging_report.md` — Детальный аналитический отчет (Root Cause Analysis) с разбором багов на основе логов LangSmith.
* `Скриншоты` — Визуальные доказательства (span-деревья) из дашборда LangSmith.
